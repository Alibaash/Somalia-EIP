## Overview

Somalia EIP is a real-time economic intelligence platform built with Streamlit and SQLite. It monitors key economic indicators across Somalia using publicly available datasets and automated data ingestion pipelines.

## Features

* Exchange Rate Analytics
* Fuel Price Monitoring
* Economic Risk Dashboard
* Market Intelligence Reporting
* Historical Regime Separation
* Automated Data Ingestion
* Real-Time Streamlit Dashboard
* SQLite Data Storage
